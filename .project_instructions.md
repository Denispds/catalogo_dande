## Deep Links & Collection Routes (Última Atualização - 2026-04-21)
- **Product Deep Link**: `/?produto=CODIGO` - auto-abre produto com lightbox
- **Collection Deep Link (legacy)**: `/?colecao=ID` - redireciona para `/colecoes/[slug]`
- **Collection Fixed Routes**: `/colecoes/[slug]` - página dedicada por coleção
  - Campo `slug` (nullable, unique) no model `CatColecao`, auto-gerado via `lib/slug.ts`
  - API: `GET /api/colecoes/by-slug?slug=xxx`
  - Server: `app/colecoes/[slug]/page.tsx` (SEO metadata dinâmico)
  - Client: `app/colecoes/[slug]/_components/collection-catalog-client.tsx`
  - Features: breadcrumbs, busca dentro da coleção, ordenação, layout toggle, seleção/export, WhatsApp share, copiar link
- **WhatsApp Messages**: Usa slug URL quando disponível (`/colecoes/aco-colares`)
- **Clipboard**: Fallback com `document.execCommand('copy')` para funcionar em iframes
- **Fluxo Vendedor**: Copia link ou WhatsApp → Cliente clica → Cai na página dedicada da coleção
