# Catálogo Digital Dande Acessórios

## Overview
Catálogo digital de acessórios para a marca Dande, migrado do Supabase para Abacus AI PostgreSQL.

## Tech Stack
- Next.js 14 (App Router), React 18, TypeScript
- Prisma ORM with PostgreSQL
- NextAuth.js v4 (credentials provider)
- AWS S3 for image uploads
- Tailwind CSS, Framer Motion, Lucide React, Sonner

## Brand Identity
- **Primary color**: #E91E8C (rosa magenta) → CSS var `--primary: 330 85% 52%`
- **Accent**: #25D366 (verde WhatsApp)
- **Font**: Inter
- **Brand name**: Dande Acessórios

## Database Schema (12 tables, prefix cat_)
- cat_departamentos, cat_categorias, cat_subcategorias
- cat_produtos (codigo PK), cat_imagens
- cat_colecoes, cat_colecao_produtos (composite PK)
- cat_acoes, cat_lancamentos, cat_links, cat_pagamentos, cat_procedimentos
- users (auth)

## Data
- 4,772 unique products (4,770 from Supabase + 2 SQL-only)
- 15 departamentos, 43 categorias
- 3 coleções, 27 imagens, 3 ações, 3 lançamentos
- 6 links úteis, 5 formas pagamento, 5 procedimentos

## Pages
- `/` - Catálogo principal (busca, filtros, grid/list, seleção, share)
- `/colecoes` - Coleções públicas com share WhatsApp
- `/central` - Central do Vendedor (ações, lançamentos, links, pagamentos, procedimentos)
- `/admin` - Painel admin protegido (CRUD produtos, coleções, upload imagens)
- `/login` - Login/signup

## Key Features
- WhatsApp sharing (com/sem preço, código, marca)
- Dark mode via next-themes
- Privacy toggles (show/hide price, code)
- Image upload via S3 presigned URLs
- Mobile-first design with bottom navigation
- Products without images are hidden by default (shown only when user searches by code/name)
- Bottom sheet filters for mobile
- Premium UI: rounded-3xl cards, smooth transitions, gradient accents
